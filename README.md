# 🚑 Ambulance Detection MVP

## 📌 Objective
To detect emergency vehicles and prioritize traffic signals using computer vision.

## ⚙️ Technologies Used
- Python
- OpenCV
- YOLOv8

## 🚀 Features
- Real-time object detection
- Emergency vehicle identification
- Automatic signal control (RED/GREEN)

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py
